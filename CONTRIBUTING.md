# 🤝 Contributing to Telegram Detective

We welcome contributors who want to help improve Telegram Detective — an AI-powered forensic tool for digital communications.

---

## 🚀 How to Contribute

1. **Fork the repository**
2. **Create a new branch**: `git checkout -b fix/my-feature`
3. **Commit your changes**: `git commit -am 'Add new feature'`
4. **Push to the branch**: `git push origin fix/my-feature`
5. **Open a Pull Request**

---

## 📂 Contribution Areas

- CLI features (Typer, argparse)
- OCR improvement and language support
- Markdown/PDF templates
- FastAPI backend or Telegram bot
- Localization (i18n)
- Writing documentation / tutorials
- Test coverage (Pytest)

---

## 📜 Code Style

- Python 3.10+
- Black for formatting (`black .`)
- Type hints and docstrings preferred
- Write meaningful commit messages

---

## 📧 Communication

Open issues freely or reach out to the maintainer at **you@example.com**
